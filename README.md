# RBAC Authentication System

A Python-based authentication system that implements Role-Based Access Control (RBAC) to manage user permissions securely.

RBAC is widely used in enterprise applications to control user access to resources based on assigned roles.

---

## Overview

Role-Based Access Control (RBAC) is a security model that restricts system access based on user roles.

This project demonstrates a simple RBAC implementation where users are assigned roles and each role has specific permissions.

---

## Features

- User authentication
- Role-based permission control
- Secure access management
- Simple command-line interface
- Easy integration into other systems

---

## Roles Example

Example roles implemented in the system:

| Role | Permissions |
|-----|-------------|
| Admin | Full access |
| Manager | Read / Write |
| User | Read only |
